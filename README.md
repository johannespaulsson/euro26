# Eurovision 2026 - Röstningsapplikation

En interaktiv webbapplikation för att rösta på bidrag i Eurovision Song Contest 2026.

## Funktioner
- **Realtidsröstning:** Använder Firebase Realtime Database för att synkronisera röster live.
- **Poängsystem:** Klassiskt Eurovision-format (1, 2, 3, 4, 5, 6, 7, 8, 10, 12 poäng).
- **Admin-panel:** Se liveresultat med lösenordsskyddad åtkomst.
- **Responsiv design:** Fungerar lika bra på mobilen som på datorn.

## Teknologier
- HTML5 / CSS3 (Montserrat font)
- Vanilla JavaScript (ES Modules)
- Firebase (Realtime Database)
